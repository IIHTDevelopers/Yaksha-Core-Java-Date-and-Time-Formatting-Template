Part 2: Date and Time Formatting and Parsing
Assessment Tasks
The next set of tasks involves formatting and parsing date and time values using DateTimeFormatter and working with different formats.

Task 1: Date Formatting
Format the current date and time into a specific format using DateTimeFormatter.

Example: Format the current date as dd/MM/yyyy and time as HH:mm:ss.

Task 2: Parse Date Strings
Parse a date string in a given format into a LocalDateTime object.

Example: Parse the string 2025-01-02 12:30 PM into a LocalDateTime.

Task 3: Time Zone Formatting
Format a date and time with respect to a specific time zone.

Example: Format a ZonedDateTime object to yyyy-MM-dd HH:mm:ss Z with a time zone.
